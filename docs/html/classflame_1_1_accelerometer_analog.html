template<ADCChannel ADCChannelX, ADCChannel ADCChannelY, ADCChannel ADCChannelZ, ADCReference ADCReference>
class flame::AccelerometerAnalog< ADCChannelX, ADCChannelY, ADCChannelZ, ADCReference >

An accelerometer that presents its data as analog values to be read by the AVR's ADC channels
