template<MHV_DECLARE_USART(usart), uint32_t baud, uint8_t rxBufLength, uint8_t txBuffers>
class mhvlib::HardwareSerial<, baud, rxBufLength, txBuffers >

Create now serial port driver

Template Parameters:
usartthe serial port parameters
baudthe baud rate to run at
rxBufLengththe maximum number of characters to receive
txBuffersthe number of send buffers
Postcondition:
Interrupts should be assigned to the driver
