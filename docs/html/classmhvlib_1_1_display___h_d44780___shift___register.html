A class for operating HD44780 based LCD displays via a shift register such as a 74HC164

Template Parameters:
colsthe number of columns
rowsthe number of rows
txBuffersthe number of output buffers
clock...the clock pin
data...the data pin (should be on the same port as clock)
enable...the enable pin
