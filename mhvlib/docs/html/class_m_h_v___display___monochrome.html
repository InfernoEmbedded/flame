template<uint16_t cols, uint16_t rows, uint8_t txBuffers>
class MHV_Display_Monochrome< cols, rows, txBuffers >

A monochrome bitmap display Origin (0,0) is bottom left

Template Parameters:
colsthe number of columns
rowsthe number of rows
txBuffersthe number of output buffers
